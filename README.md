# AI Smart Expense Tracker

An AI-powered Expense Tracker built using Python and Streamlit that helps users manage daily expenses, analyze spending habits, and gain smart financial insights through interactive dashboards and visual reports.

---

## Features

- Add and manage daily expenses
- Categorize transactions
- Monthly and weekly expense analysis
- Budget planning and tracking
- Interactive charts and analytics
- AI-generated spending insights
- Expense history management
- Responsive and modern UI
- CSV report export

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- SQLite
- CSS

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Smart-Expense-Tracker.git
```

### Move into Project Folder

```bash
cd AI-Smart-Expense-Tracker
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
streamlit run main.py
```
